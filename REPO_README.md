## 🚀 Featured Projects

### 🇪🇺 EU AI Act Risk Classifier
**`projects/eu_ai_act_classifier.ipynb`**

A Python-based Jupyter notebook that classifies any AI use case into its EU AI Act risk tier — Unacceptable, High, Limited, or Minimal Risk.

**Features:**
- Rule-based keyword classifier covering all four EU AI Act risk categories
- Full compliance obligation output per classification
- Applicable article references (Articles 5–95)
- Recommended next steps for each risk level
- Batch classification mode for multiple use cases
- Interactive and hardcoded input modes

**Run it:**
```bash
jupyter notebook projects/eu_ai_act_classifier.ipynb
```

### 🎗️ Breast Cancer AI Classifier
**`projects/breast-cancer-classifier.ipynb`**

A machine learning classifier built to detect malignant vs. benign breast cancer cases using the Wisconsin Breast Cancer Dataset. Demonstrates responsible AI development practices including model evaluation, bias consideration, and clinical risk context.
